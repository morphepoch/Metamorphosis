# Metamorphosis 🦋

**AI & Aesthetics** code-and-thought space.

Metamorphosis is a repository for my ongoing explorations at the intersection of AI, expression, and form.

Here, I treat code not only as a functional tool, but also as a medium for rhythm, structure, and beauty.

## What Metamorphosis Explores

At this stage, Metamorphosis focuses on three directions:

- **AI & verbal aesthetics**  
  How AI can shape language, release notes, announcements, writing systems, and product communication with more clarity, rhythm, and texture.

- **AI & visual aesthetics**  
  How AI can work with layout, interface, imagery, and visual expression to create more sensitive and compelling experiences.

- **AI & logical aesthetics**  
  How structure itself can be beautiful: prompt design, context organization, harness engineering, and the elegance of well-shaped systems.

More subfolders and subcategories will be added over time as the repository grows.

## Current Project

The current uploaded skill in this repository is:

- **`releasenote-writing`**

It is a writing skill for turning internal product notes into polished user-facing release notes and product announcements.

It belongs to the **verbal aesthetics** direction and currently lives at:

- `releasenote-writing/`

# releasenote-writing

`releasenote-writing` is a verbal aesthetics skill inside **Metamorphosis**.

It focuses on turning internal product notes, changelog items, PRDs, launch docs, and rough bullets into polished user-facing release notes and product announcements.

## Included Files

- `SKILL.md`: core skill definition
- `references/style-guide.md`: style references
- `agents/openai.yaml`: agent configuration

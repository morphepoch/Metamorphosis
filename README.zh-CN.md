# Metamorphosis 🦋

**AI 与美学** 的代码与思考空间。

Metamorphosis 是我持续进行中的一个探索仓库，关注 AI、表达与形式之间的交叉地带。

在这里，我把代码视为一种不只服务功能的媒介，也把它看作节奏、结构与美感的载体。

## Metamorphosis 正在探索什么

现阶段，Metamorphosis 主要聚焦三个方向：

- **AI 与文字美学**  
  探索 AI 如何作用于语言、发布说明、公告、写作系统和产品表达，让它们拥有更清晰的结构、更自然的节奏和更好的质感。

- **AI 与视觉美学**  
  探索 AI 如何与布局、界面、图像和视觉表达协作，创造更敏感、更有吸引力的体验。

- **AI 与逻辑美学**  
  探索结构本身的美感：prompt design、context organization、harness engineering，以及被精心组织起来的系统之优雅。

随着仓库继续生长，后续也会逐步增加更多子目录和分类。
